# Dataværktøjer

Desktop-program til undervisningsteamet i Den Gamle By, der automatiserer de mest tidskrævende Excel-opgaver.

## Hvad kan det?

- **Excel Import** – trækker data ud af Word-dokumenter og skriver dem direkte ind i årsarket i Excel
- **Statistik Maker** – genererer statistik-tabeller på baggrund af importeret data
- **Skabelon Maker** – opretter nye årsark med korrekt struktur klar til brug

## Installation

Du behøver ikke installere noget. Download den nyeste `.exe`-fil under [Releases](../../releases) og kør den direkte.

## Til udviklere

Kræver Python 3.11+.

```bash
pip install -r requirements.txt
python main.py
```
